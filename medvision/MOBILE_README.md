# 📱 Мобильная адаптация интерфейса MedVision AI

## Обзор

Интерфейс приложения MedVision AI полностью адаптирован для мобильных устройств, планшетов и десктопов. Приложение обеспечивает оптимальный пользовательский опыт на всех размерах экранов.

## 🎯 Особенности адаптации

### 📱 Мобильные устройства (< 768px)
- **Компактное меню**: Выдвижное боковое меню с гамбургер-иконкой
- **Оптимизированные таблицы**: Преобразованы в карточки для удобства чтения
- **Полноширинные кнопки**: Легко нажимать на сенсорных устройствах
- **Адаптивный шрифт**: Размеры 13-16px для легкого чтения
- **Минимальные отступы**: 8-16px для экономии места

### 📲 Планшеты (768px - 1024px)
- **Сжатое меню**: Боковая панель 80px с иконками
- **Двухколоночные таблицы**: Компромисс между компактностью и читаемостью
- **Оптимизированные сетки**: Адаптивные размеры контейнеров

### 🖥️ Десктоп (> 1024px)
- **Полноразмерное меню**: Боковая панель 240px со всеми лабелами
- **Таблицы полноразмерные**: Стандартное отображение
- **Максимальная функциональность**: Все элементы видны одновременно

## 🔧 Технические детали

### CSS медиа-запросы
```css
/* Планшеты */
@media (max-width: 1024px) { ... }

/* Мобильные */
@media (max-width: 767px) { ... }

/* Маленькие экраны */
@media (max-width: 480px) { ... }
```

### JavaScript функции
- `initMobileMenu()` - инициализация мобильного меню
- `renderHistory()` - вывод истории с data-label
- `renderRecent()` - вывод последних анализов с data-label

### Touch-оптимизация
- Минимальный размер кликабельного элемента: **44×44px** (рекомендация Apple)
- Быстрая прокрутка на iOS: `-webkit-overflow-scrolling: touch`
- Поддержка Safe Area для iPhone X и новее
- Отключение zoom при вводе текста: `font-size: 16px`

## 🚀 Использование

### Для разработчика
Адаптация не требует специальной конфигурации. Просто запустите приложение, и оно автоматически подстроится под размер экрана.

### Для пользователя
1. **Мобильное меню**: Нажмите ☰ для открытия/закрытия меню
2. **Загрузка файлов**: Перетащите файл или нажмите на зону загрузки
3. **Таблицы**: Прокрутите горизонтально для просмотра всех столбцов
4. **Результаты**: Все элементы оптимизированы для сенсорного ввода

## 📝 Форматирование таблиц на мобильных

На мобильных устройствах таблицы автоматически преобразуются в карточки с использованием CSS свойства `display: block`. Каждая ячейка получает данные из атрибута `data-label`:

```html
<td data-label="Диагноз:"><span class="badge green">Норма</span></td>
```

На мобильных отображается как:
```
Диагноз: Норма
```
