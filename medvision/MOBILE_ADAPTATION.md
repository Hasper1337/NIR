# Адаптация интерфейса для мобильных устройств

## Описание изменений

Интерфейс приложения MedVision AI был полностью адаптирован для мобильных устройств и планшетов.

## Что было изменено:

### 1. **HTML (`templates/index.html`)**
- Добавлена кнопка меню (`menu-toggle`) для мобильных устройств
- Обёртка навигации теперь имеет ID `sidebar` для управления из JavaScript
- Текст навигации обёрнут в `<span class="nav-label">` для скрытия на мобильных

### 2. **CSS (`static/style.css`)**

#### Touch-оптимизация
- Минимальные размеры кликабельных элементов: 44×44 px (рекомендация Apple)
- Улучшена прокрутка на iOS: `-webkit-overflow-scrolling: touch`
- Поддержка Safe Area для iPhone с notch
- Отключено масштабирование шрифтов: `font-size: 16px` для input на мобильных

#### Адаптивные макеты

**Планшеты (1024px и выше)**
- Сжатая боковая панель (80px) с укрытыми лабелами
- Одноколоночные таблицы

**Мобильные устройства (767px и меньше)**
- Горизонтальное меню в шапке (фиксировано)
- Выдвижное боковое меню (hamburger)
- Таблицы преобразованы в карточки (display: block)
- Полноширинные кнопки
- Оптимизированные размеры шрифтов
- Безопасные области для iOS

**Маленькие экраны (480px и меньше)**
- Уменьшены отступы и размеры
- Минимизирована высота элементов
- Улучшена читаемость текста

### 3. **JavaScript (`static/script.js`)**

#### Мобильное меню
- Функция `initMobileMenu()` управляет открытием/закрытием меню
- Автоматическое закрытие при клике на пункт меню
- Закрытие при клике вне меню

#### Таблицы на мобильных
- Добавлены `data-label` атрибуты для отображения заголовков строк
- Таблицы отображаются как стопка карточек
- Каждое поле показывает свой заголовок

## Ключевые особенности

✅ **Полная поддержка сенсорных устройств**
- Увеличенные области нажатия
- Отключены hover-эффекты на сенсорных устройствах
- Улучшена обратная связь при нажатии

✅ **Адаптивный дизайн**
- 3 точки разрыва: 1024px, 767px, 480px
- Гибкие сетки и макеты
- Оптимизированные размеры шрифтов

✅ **Оптимизация производительности**
- Минимальные медиа-запросы
- Эффективная прокрутка
- Быстрые анимации (0.3s)

✅ **Совместимость**
- Chrome, Firefox, Safari (включая iOS)
- Edge на Windows
- Все современные мобильные браузеры

## Тестирование

Рекомендуется протестировать на:
- iPhone 12, iPhone 14 Pro Max
- iPad (все модели)
- Samsung Galaxy (различные размеры)
- Chrome DevTools Device Mode

## Использование

Интерфейс автоматически адаптируется на основе размера экрана. Никаких дополнительных действий от пользователя не требуется.

### Мобильное меню
На мобильных устройствах:
1. Нажмите ☰ (гамбургер меню)
2. Выберите нужный раздел
3. Меню автоматически закроется

## Метрики доступности

- **Размер текста**: 13-16px на мобильных (легко читается)
- **Контрастность**: WCAG AA
- **Кликабельные элементы**: min 44×44px
- **Отступы**: 8-16px на мобильных
